# Daily Drawing Fundamentals Trainer

Idea-based project scaffold for:
`https://www.ideabrowser.com/idea/daily-drawing-fundamentals-trainer`

## Product Snapshot
- Core idea: short daily drawing lessons for beginners/hobbyists
- Differentiator: AI feedback on uploaded sketches
- Habit loop target: users complete lesson 3 in week 1
- Monetization: $5/month or $50/year (B2C), phase-2 B2B licensing path

## Stack Baseline
- Next.js + TypeScript + shadcn/ui
- Convex + Clerk
- TanStack Query + TanStack Router
- Jest (unit/integration)
- Playwright (e2e)
- Vercel deployment

## Scripts
```bash
bun dev
bun run lint
bun run test
bun run test:e2e
bun run build
```

## Notes
- This repository currently contains scaffold + planning artifacts for implementation handoff.
- No production business logic is implemented yet.
